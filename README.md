# java-web-gradle-spring-thyme-dolt-simple

## Description
A thyme springboot java gradle build,
that connects to dolt database.

## Tech stack
- springboot
  - web
- thymeleaf
- gradle
  - 6.8.2
- bootstrap
- jquery
- dataTable

## Docker stack
- dolthub/dolt-sql-serverdb:latest
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
